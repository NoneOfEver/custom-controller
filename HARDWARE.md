# Custom Controller — 硬件外设规格（可落地版）

> 目的：作为后续 Rust 工程初始化与外设驱动实现的唯一依据。
> 说明：位置传感器与上位机的**协议细节由你后续补充**，本文已把“接口数量、引脚、串口参数、分帧/接收方式”等与代码强相关的部分固定下来。

## 1. MCU / 供电 / 时钟 / 调试

- MCU：STM32F446RET6
- 供电：MCU 3.3V 供电；无电源监测（BOR）需求
- 时钟：HSE=8MHz；无 LSE
- 系统主频：配置为该 MCU 规格允许的最大值（后续按芯片规格落地）
- 调试与下载：SWD 固定引出；不需保留 BOOT0/串口下载

## 2. 外设清单（数量固定）

| 外设 | 数量 | 接口 | 用途 | 关键约束 |
|---|---:|---|---|---|
| 位置传感器 | 4 | UART/USART | 持续推送数据 | TTL 电平；`9600 8N1`；DMA + IDLE 分帧 |
| 上位机通讯 | 1 | USART1 → RS485 | 向上位机发送数据 | **仅发送 TX**；不接收；无需 DE/RE |
| 摇杆三轴 | 1 | ADC1（3 通道） | 读取 3 轴模拟量 | 输入 0–3.3V；通道固定 IN8/IN4/IN5 |
| 摇杆按键 | 1 | GPIO | 数字输入 | 上拉、低有效；周期扫描 + 20ms 去抖 |
| 自锁按键 | 4 | GPIO | 数字输入 | 上拉、低有效；周期扫描 + 20ms 去抖；功能=模式 |

## 3. 引脚分配（已落地）

### 3.1 UART/USART（共 5 路）

通用串口参数（5 路一致）：

- 物理电平：TTL
- 串口参数：`9600 8N1`
- 数据流：持续推送（协议/帧格式后续由你补充）
- 分帧：空闲中断（IDLE）
- 接收实现：DMA + IDLE（RS485 上位机通道不接收）

| 通道用途 | 外设实例 | TX 引脚 | RX 引脚 | 备注 |
|---|---|---|---|---|
| RS485 上位机（仅发送） | USART1 | PA9 | PA10 | 仅使用 TX；RX 在软件层不启用 |
| 位置传感器 #1 | UART4 | PA0 | PA1 | DMA RX + IDLE |
| 位置传感器 #2 | USART2 | PA2 | PA3 | DMA RX + IDLE |
| 位置传感器 #3 | USART3 | PC10 | PC11 | DMA RX + IDLE |
| 位置传感器 #4 | USART6 | PC6 | PC7 | DMA RX + IDLE |

### 3.2 ADC（摇杆三轴，ADC1）

- 输入电压：0–3.3V
- 通道与引脚：

| 轴（逻辑名） | ADC 通道 | 引脚 | 备注 |
|---|---|---|---|
| Axis-A | ADC1_IN8 | PB0 | 采样顺序第 1 |
| Axis-B | ADC1_IN4 | PA4 | 采样顺序第 2 |
| Axis-C | ADC1_IN5 | PA5 | 采样顺序第 3 |

- 采样方式：软件触发 + 多通道扫描
- 扫描顺序：IN8 → IN4 → IN5
- 采样时间：待定（你可能后续会改）
- 分辨率/参考：12-bit；VDDA=3.3V

### 3.3 按键 GPIO（共 5 个）

按键电气与输入约束：

- 逻辑：按下为 0V（低电平有效）
- 上拉：板级已做上拉；MCU 侧配置为输入、`no-pull`（不启用内部上拉/下拉）
- 采集：周期扫描（不使用 EXTI）
- 去抖：20ms

| 按键 | 引脚 | 端口 |
|---|---|---|
| BUTTON1 | PC0 | GPIOC |
| BUTTON2 | PC1 | GPIOC |
| BUTTON3 | PC2 | GPIOC |
| BUTTON4 | PC3 | GPIOC |
| JOYSTICK_BUTTON | PB7 | GPIOB |

#### 4.2.1 临时最小上报帧（已在固件中实现，用于联调）

说明：为尽快打通“板端 → 上位机”的联调链路，固件当前实现了一个**固定长度、无堆分配**的输入状态上报帧；后续你可以替换为正式协议。

- 发送通道：USART1（TX-only）
- 发送节奏：20ms/帧（50Hz）
- 端序：Little-endian
- 帧长度：12 bytes

字节布局：

| Byte Offset | 字段 | 类型 | 说明 |
|---:|---|---|---|
| 0..1 | magic | u8[2] | 固定 `0xA5 0x5A` |
| 2 | ver | u8 | 版本号（当前固定 `0x01`） |
| 3 | seq | u8 | 包序号（0..255 循环） |
| 4..5 | axis_a | u16 | 摇杆 Axis-A 原始 ADC（12-bit，落在 u16） |
| 6..7 | axis_b | u16 | 摇杆 Axis-B 原始 ADC |
| 8..9 | axis_c | u16 | 摇杆 Axis-C 原始 ADC |
| 10 | buttons_mask | u8 | 按键位图（1=Pressed） |
| 11 | checksum | u8 | `XOR(out[2]..out[10])` |

`buttons_mask` 位定义（bit=1 表示按下；按键为低有效，但这里已转换为逻辑态）：

| Bit | 按键 |
|---:|---|
| 0 | BUTTON1 (PC0) |
| 1 | BUTTON2 (PC1) |
| 2 | BUTTON3 (PC2) |
| 3 | BUTTON4 (PC3) |
| 4 | JOYSTICK_BUTTON (PB7) |
